# 💰 AI Finance Advisor

An AI-powered finance tracking app that helps users manage transactions and receive smart financial advice.

## 🚀 Features
- User authentication (JWT)
- Add & track transactions
- AI-powered financial advice (Groq API)
- Full-stack (React + Node.js + MongoDB)

## 🛠️ Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- AI: Groq API

## 📦 Installation

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm start
